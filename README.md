# course3_week4_project

This repository contains 3 files:
1. run_analysis.R
2. CodeBook.md
3. README.md

To run the script, open run_analysis.R in RStudio and run the script, which will perform the following tasks:
1. Download and unzip the dataset and set the necessary working directory for the script to work
2. Merge training and test data sets 
3. Extract only the measurements on the mean and standard deviation
4. Use descriptive activity names to name the activities in the data set
5. Appropriately labels data set with descriptive variable names
6. Creates a second, independent tidy data set with the average of each variable for each activity and each subject
